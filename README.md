## Payment Tier by Gender

![Payment Tier by Gender](https://github.com/user-attachments/assets/22821e5d-d0a5-4825-9f2c-7e14202f16a8)

This dot plot displays employee payment tiers separated by gender. Each dot represents an individual employee, allowing for a comparison of how payment tiers are distributed between male and female employees. This visualization helps identify potential patterns or disparities in compensation levels across genders.

The original dataset was obtained from Kaggle:  
https://www.kaggle.com/
